# IPL-2022-Data-Analysis-Using-Python
A complete data analysis of the IPL 2022 season using Python. This project explores match outcomes, toss decisions, top scorers, and team performances. Visualizations are created with Matplotlib, Seaborn, and Plotly to provide both static and interactive insightswith the help of python libraries.

# Files in Repository

IPL 2022 Data-Analysis.ipynb → Colab Notebook with all the analysis and visualizations.

IPL 2022.csv → Dataset containing IPL 2022 match details.

# Key Analysis Performed

Team that wins by the highest run margins / successful chases

Match-winning trends after winning the toss

Most successful IPL teams

Top scorer in IPL 2022

Team-wise most likely toss decisions after winning toss.

Interactive and static visualizations using Plotly, Seaborn, and Matplotlib.

# Tech Stack

Python (Libraries: Pandas, NumPy, Matplotlib, Seaborn, Plotly).
